https://github.com/SnipMeDev/Highlights

## Features
- 17 supported languages (Kotlin, Dart, Swift, PHP, etc)
- Light / dark mode
- 6 built-in themes
- Phrase bolding (emphasis)
- Result caching and support for incremental changes

## Support

- Android ✅
- iOS ✅
- macOS 🔴 (Not yet)
- Linux 🔴 (Not yet)
- Windows 🔴 (Not yet)
- Web 🔴 (Not yet)
